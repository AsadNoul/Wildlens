# WildLens

WildLens is a mobile application that allows users to identify animals using their device's camera. Users can also explore a gallery of animals, learn about their habitats, and save their favorite animals to a personal collection.

## Features

* **Animal Identification:** Identify animals by taking a photo or selecting one from the gallery.
* **Animal Details:** View detailed information about animals, including their taxonomy, characteristics, and habitat.
* **Animal Gallery:** Browse a gallery of animal images.
* **Collection:** Save your favorite animals to a personal collection.
* **Explore:** Search for animals and filter by category.

## Getting Started

To get started, you'll need to create a `.env` file in the root of the project and add the following API keys:

```
API_NINJAS_KEY=YOUR_API_KEY_HERE
UNSPLASH_ACCESS_KEY=YOUR_API_KEY_HERE
ANIMAL_DETECT_KEY=YOUR_API_KEY_HERE
ANIMAL_DISCOVERY_KEY=YOUR_API_KEY_HERE
```

Once you've added the API keys, you can run the app using the following commands:

```
npm install
npm run android
```
